# study_planner

Study Planner is a clean and intuitive productivity app designed to help students organize and track their daily study goals. The app features a modern dashboard layout with a SliverAppBar for dynamic scrolling effects, a prominent “Today’s Focus” card to highlight key subjects or goals, and a scrollable task list for managing study sessions efficiently. With its elegant UI and simple navigation, Study Planner provides a smooth user experience, combining functionality and design to boost focus and productivity throughout the day.

# Study Planner

A clean and minimal Flutter dashboard to help you stay organized and focused.  
It features a sliver app bar, a stacked **“Today’s Focus”** card, and a scrollable list of study tasks — built for smooth daily planning and productivity.

---

## Features

- **Today’s Focus Card** — Highlights your main goal or subject for the day.  
- **Sliver App Bar** — Smooth and modern scrolling experience.  
- **Scrollable Task List** — Add, view, and manage your study sessions easily.  
- **Simple UI** — Minimal design that keeps you focused on what matters.  

---

## Built With

- **Flutter** — Beautiful native apps in one codebase.  
- **Dart** — Modern, expressive, and fast.
